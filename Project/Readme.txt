How to run Parallel_Bucketsort_via_OpenMP.c
1. Download Parallel_Bucketsort_via_OpenMP.c
2. type "gcc Parallel_Bucketsort_via_OpenMP.c -o para -fopenmp"
3. type "./para"
4. Enter input to select the length of the arrays that have 1,2,3
5. Enter input to select the number of threads/buckets that have 1,2,3,4,5
